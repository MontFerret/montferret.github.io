- **Getting Started** covers installation and first useful scripts.
- **Language** explains the FQL concepts you need before reading reference entries.
- **Web Extraction** focuses on page querying, output, and error handling.
- **Tools** documents the Ferret CLI, Lab, Worker, Playground, Docker images, and editor integrations.
- **Reference** contains detailed statement and API entries.
