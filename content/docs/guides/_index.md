# Guides

Guides walk through practical workflows for extracting data, automating browsers, handling errors, and integrating Ferret into applications. Each guide builds on the concepts from the [language reference]({{< ref "/docs/language" >}}) and shows how to combine them in real scripts.

If you are new to Ferret, start with the [Quick Start]({{< ref "/docs/getting-started/quick-start" >}}).
