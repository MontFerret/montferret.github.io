## New Website, New Docs, and the Road to Ferret v2

Hello friends,

You might have noticed that Ferret has a refreshed website and a new domain name. This is part of the ongoing work around Ferret v2, and the website is gradually becoming the main home for documentation, guides, and project updates.

One of the main focuses right now is improving the docs. Ferret v2 brings a cleaner language model, new runtime concepts, better tooling, and a more structured ecosystem, so the documentation needs to explain not only the syntax, but also how the pieces fit together.

The docs are still in flux. Some sections are already available, while others are being rewritten, expanded, or moved from older sources into the new website. The goal is to make the website the single source of truth for Ferret going forward.

This work is happening alongside active development of Ferret v2 itself. The language, CLI, debugger, modules, and supporting tools are still evolving, so some pages may continue to change as the project moves closer to beta.

In short: the new website is live, the docs are growing, and Ferret v2 is steadily taking shape.

Stay tuned for more updates!
